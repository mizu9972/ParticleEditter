[概要]
パーティクルシステムのエディターです。
Windows環境でのみ動作を確認しています。
ImGuiを使ってパーティクルシステムの設定情報を表示しており、
数値を変更するとそのまま描画されているパーティクルに反映されます。

設定したパーティクルシステムのデータはファイル出力でき、
パーティクルシステムのクラスを移植したプログラムで、そのデータを利用して
自作のパーティクルシステムを流用することができます。

パーティクルはゲームの表現力に大きく貢献すると考えており、
パーティクルシステムの設定がスムーズにできれば
それだけ多くのこだわりのあるパーティクルを実装できるので、
ゲームのクオリティを上げることが少しでも容易になればと思い作成しました。

[画面]
画面左側のウィンドウは、パーティクルシステムの設定です。
画面右側のウィンドウは、ファイル入出力の操作部分になっています。
基本的にマウスでの操作になります。

[ファイル入出力]
Exportボタンを押すと、プロジェクト内のOutPutDataフォルダ内に
設定したパーティクルシステムのデータが出力されます。

ImportはInPutDataフォルダ内のファイルが、
パーティクルシステムのデータと割り当てるテクスチャがそれぞれ別々のツリーノード内に
表示され、選択すると選んだ物を反映させられます。
